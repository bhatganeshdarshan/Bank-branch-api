Bank Branch API Documentation

Base URL

http://localhost:5000/

1. Home Page

URL: /

Method: GET

Description: Returns the homepage of the API. This is a basic HTML page.

2. Get All Bank Branches

URL: /all

Method: GET

Description: Retrieves a list of all bank branches stored in the database.

Response:

{
    "banks": [
        {
            "ifsc": "ABCD1234567",
            "bank_id": 1,
            "branch": "Example Branch",
            "address": "123 Main Street",
            "city": "Example City",
            "district": "Example District",
            "state": "Example State",
            "bank_name": "Example Bank"
        },
        // ... (more branches)
    ]
}

3. Search for Bank Branches by Location

URL: /search?loc=city_name

Method: GET

Parameters:

loc (query parameter): The location or branch name to search for.

Description: Searches for bank branches by location or branch name and returns the matching results.

Response (200 OK):

{
    "banks": [
        {
            "ifsc": "ABCD1234567",
            "bank_id": 1,
            "branch": "Example Branch",
            "address": "123 Main Street",
            "city": "Example City",
            "district": "Example District",
            "state": "Example State",
            "bank_name": "Example Bank"
        },
        // ... (more matching branches)
    ]
}

Response (404 Not Found):

{
    "error": {
        "error": "Did not find any banks at that location !!"
    }
}

Error Handling

- If there are any errors in the API request, appropriate HTTP status codes and error messages will be returned.

- Common error status codes include:

404 (Not Found): When searching for branches and no matching branches are found.

500 (Internal Server Error): If there are internal server errors or issues with the database.
